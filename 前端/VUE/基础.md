| 语法 | 用途 |
| :----: | :----: |
| `{{ state }}` | 将**文本内容**与 JS 状态绑定 |
| `:[attribute]="[state]"` | 将标签**属性**与 JS 状态绑定 |
| `@[event]="[handler]"` | 将标签**事件**与相应处理绑定 |
| `v-model="[state]"` | 将标签内部自带的状态(如表单控件 `input` 的 `value` 属性)与 JS 状态**双向绑定** |
| `v-if="[state]"` | 基于 JS 状态的 `Boolean` 性质, **条件渲染**该标签 |
| `v-for="[element in elements]"` | 基于可遍历 JS 状态的各项数值, **批量渲染**标签(需要用 `:key` 对各个元素进行唯一性标注) |
